To compile project:

- clone git to local folder: git clone https://github.com/alonabas/PostPCProject.git .
- put tag v1: git checkout v1

If you use eclipse: add both projects: android-support-v7-appcompat and HujiMe,

In HujiMe there are files with Hebrew strings, to see them you need to change encoding:

select HujiMe project properties -> in Resource (first section) select Text file encoding to Other ->
in Other select UTF-8 ->recompile
